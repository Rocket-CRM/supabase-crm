## Functional Feature: Brand.com (1st Party E-Commerce) — Corporate License Brand.com using Shopify

### (1) Shopify License (Monthly Recurring)

Shopify plan license fee (Basic / Shopify / Advanced — select applicable tier).

- **Basic** — $39 USD/mo, transaction fee 2.9% + 30¢, up to 2 staff accounts
- **Shopify** — $105 USD/mo, transaction fee 2.7% + 30¢, up to 5 staff accounts
- **Advanced** — $399 USD/mo, transaction fee 2.5% + 30¢, up to 15 staff accounts, custom reports

*Shopify license is billed directly by Shopify to the client. Rocket facilitates setup and ongoing management.*

---

### (2) One-Time Shopify Store Activation & Setup

One-time setup fee for store launch, including:

- Store provisioning and initial configuration
- Theme selection and application (from Shopify Theme Store — non-custom)
- Theme configuration: logo, brand colors, typography, homepage layout
- Navigation and menu structure setup
- Product creation and import (initial catalog upload)
- Product category and collections setup
- Payment gateway integration (Stripe / Omise — credit cards, PromptPay, TrueMoney)
- Shipping zones and delivery methods configuration (flat rate / weight-based)
- Tax settings for Thailand (VAT 7%)
- Email notification templates setup (order confirmation, shipping, refund)
- Custom domain connection and SSL
- Google Analytics / Meta Pixel integration
- Basic SEO setup (meta titles, descriptions, sitemap)
- Admin account provisioning and staff permissions
- UAT and go-live support

---

### (3) Shopify Loyalty Program Plugin & Integration (30% on top of Shopify License — Monthly Recurring)

Monthly fee calculated as **30% of the applicable Shopify plan license fee**, covering:

**Loyalty Widget & Storefront Integration**
- Embedded loyalty widget on Shopify storefront (member profile, points balance, tier status)
- Customer can view available rewards and redeem directly on online store
- Apply loyalty points as discount at checkout (points-to-discount conversion)
- Apply loyalty coupons/promo codes at checkout
- Automatic points earning on completed online store purchases (order webhook sync)
- Tier-based exclusive pricing or discounts (synced via Shopify customer tags)
- Member registration / login via loyalty program from storefront

**Backend Integration**
- Real-time order sync from Shopify to Rocket loyalty engine (webhook-based)
- Automatic points calculation and crediting on order fulfillment
- Points reversal on order cancellation/refund
- Coupon and promo code generation synced to Shopify discount codes
- Customer data sync between Shopify and Rocket CRM (profile, tags, tier)
- Tier upgrade/downgrade reflected in Shopify customer tags automatically

**Ongoing Maintenance**
- Integration monitoring and error handling
- Compatibility updates for Shopify platform changes
- Loyalty widget updates and bug fixes

*This item covers the Rocket loyalty plugin, storefront widget, Shopify-to-Rocket integration, and ongoing maintenance. The Shopify store features (products, checkout, shipping, blog, etc.) are covered by the Shopify license in Item 1.*
